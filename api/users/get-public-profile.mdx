---
title: Get public profile
api: GET /api/users/{id}
description: Retrieve the public profile of any user
---

## Endpoint

```text
GET /api/users/{id}
```

## Authentication

Requires a valid JWT access token in the Authorization header.

```text
Authorization: Bearer <access_token>
```

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the user
</ParamField>

## Response

Returns the user's public profile, which contains only their first and last name.

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp of the response
</ResponseField>

<ResponseField name="data" type="object">
  User public profile data
  
  <Expandable title="properties">
    <ResponseField name="firstName" type="string">
      User's first name
    </ResponseField>
    
    <ResponseField name="lastName" type="string">
      User's last name
    </ResponseField>
  </Expandable>
</ResponseField>

<Note>
  Public profiles only expose the user's name. Email, roles, and other sensitive information are not included.
</Note>

## Example Request

```bash
curl -X GET http://localhost:8080/api/users/5 \
  -H "Authorization: Bearer <access_token>"
```

## Example Response

```json
{
  "success": true,
  "timestamp": "2026-03-03T10:30:00Z",
  "data": {
    "firstName": "John",
    "lastName": "Doe"
  }
}
```
