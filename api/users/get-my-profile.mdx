---
title: Get my profile
api: GET /api/users/me
description: Retrieve the authenticated user's private profile
---

## Endpoint

```text
GET /api/users/me
```

## Authentication

Requires a valid JWT access token in the Authorization header.

```text
Authorization: Bearer <access_token>
```

## Response

Returns the authenticated user's private profile including email and roles.

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp of the response
</ResponseField>

<ResponseField name="data" type="object">
  User private profile data
  
  <Expandable title="properties">
    <ResponseField name="id" type="number">
      Unique identifier of the user
    </ResponseField>
    
    <ResponseField name="email" type="string">
      User's email address
    </ResponseField>
    
    <ResponseField name="firstName" type="string">
      User's first name
    </ResponseField>
    
    <ResponseField name="lastName" type="string">
      User's last name
    </ResponseField>
    
    <ResponseField name="roles" type="array">
      Array of user roles (e.g., ["MEMBER"], ["ADMIN"])
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X GET http://localhost:8080/api/users/me \
  -H "Authorization: Bearer <access_token>"
```

## Example Response

```json
{
  "success": true,
  "timestamp": "2026-03-03T10:30:00Z",
  "data": {
    "id": 1,
    "email": "john.doe@example.com",
    "firstName": "John",
    "lastName": "Doe",
    "roles": ["MEMBER"]
  }
}
```

## Error Responses

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    ```json
    {

      "error": "Unauthorized"
    ```
