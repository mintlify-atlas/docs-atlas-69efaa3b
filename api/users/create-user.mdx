---
title: Create user
api: POST /api/management/users
description: Create a new user account (admin only)
---

## Endpoint

```text
POST /api/management/users
```

## Authentication

Requires ADMIN role. Include a valid JWT access token in the Authorization header.

```text
Authorization: Bearer <admin_access_token>
```

## Request Body

<ParamField path="email" type="string" required>
  User's email address
  
  **Validation:**
  - Not blank
  - Valid email format
  - Must match pattern: `^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$`
  - Must be unique in the system
</ParamField>

<ParamField path="password" type="string" required>
  User's password
  
  **Validation:**
  - Not blank
  - Minimum 8 characters
</ParamField>

<ParamField path="firstName" type="string" required>
  User's first name
  
  **Validation:**
  - Not blank
  - 2-50 characters
  - Letters, spaces, periods, hyphens, and apostrophes only
</ParamField>

<ParamField path="lastName" type="string" required>
  User's last name
  
  **Validation:**
  - Not blank
  - 2-50 characters
  - Letters, spaces, periods, hyphens, and apostrophes only
</ParamField>

<ParamField path="roles" type="array" required>
  Array of role strings to assign to the user
  
  **Validation:**
  - At least one role required
  - Valid values: `MEMBER`, `ADMIN`
</ParamField>

## Response

Returns the created user's information (excluding password).

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp of the response
</ResponseField>

<ResponseField name="data" type="object">
  Created user data
  
  <Expandable title="properties">
    <ResponseField name="id" type="number">
      Unique identifier of the created user
    </ResponseField>
    
    <ResponseField name="email" type="string">
      User's email address
    </ResponseField>
    
    <ResponseField name="firstName" type="string">
      User's first name
    </ResponseField>
    
    <ResponseField name="lastName" type="string">
      User's last name
    </ResponseField>
    
    <ResponseField name="roles" type="array">
      Assigned roles
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl -X POST http://localhost:8080/api/management/users \
  -H "Authorization: Bearer <admin_access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "jane.smith@example.com",
    "password": "SecurePass123!",
    "firstName": "Jane",
    "lastName": "Smith",
    "roles": ["MEMBER"]
  }'
```

## Example Response

```json
{
  "success": true,
  "timestamp": "2026-03-03T10:30:00Z",
  "data": {
```
