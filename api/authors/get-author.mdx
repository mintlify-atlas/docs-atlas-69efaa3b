---
title: 'Get Author'
api: 'GET /api/authors/{id}'
description: 'Retrieve a specific author by their ID'
---

## Endpoint

```
GET /api/authors/{id}
```

## Authentication

Requires authentication with a valid Bearer token.

## Path Parameters

<ParamField path="id" type="long" required>
  Unique identifier of the author to retrieve
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="timestamp" type="string" required>
  ISO 8601 timestamp of the response
</ResponseField>

<ResponseField name="data" type="object" required>
  Author details
  
  <ResponseField name="id" type="long" required>
    Unique identifier for the author
  </ResponseField>
  
  <ResponseField name="name" type="string" required>
    Full name of the author
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.example.com/api/authors/1" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json"
```

## Example Response

```json
{
  "success": true,
  "timestamp": "2026-03-03T10:00:00Z",
  "data": {
    "id": 1,
    "name": "George Orwell"
  }
}
```

## Error Responses

### 401 Unauthorized

```json
{
  "success": false,
  "timestamp": "2026-03-03T10:00:00Z",
  "error": {
    "code": "UNAUTHORIZED",
    "message": "Authentication required"
  }
}
```

### 404 Not Found

```json
{
  "success": false,
  "timestamp": "2026-03-03T10:00:00Z",
  "error": {
    "code": "NOT_FOUND",
    "message": "Author not found with id: 999"
  }
}
```
